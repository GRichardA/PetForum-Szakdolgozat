# Wireframes – PetShop Események

## 01-main-list.png
**Cél:** A felhasználó látja a közelgő kutyás eseményeket.
**Interakciók:** "Új esemény" gomb, Esemény kártyák (Cím, Dátum, Helyszín).
**Állapotok:** Normál lista (van adat).
**Hivatkozások:** US-04/AC1, Spec v0.2 – Fő AC-k/2.

## 02-create-form.png
**Cél:** Új esemény meghirdetése.
**Interakciók:** Űrlap kitöltése (Cím, Dátum, Helyszín), "Mentés" gomb.
**Állapotok:** Szerkesztés.
**Hivatkozások:** US-02/AC1.

## 03-empty-state.png
**Cél:** Ha még nincs esemény, a felhasználó ne üres oldalt lásson.
**Interakciók:** Nagy "Esemény létrehozása" gomb (CTA).
**Állapotok:** Üres adatbázis.
**Hivatkozások:** US-01/AC1.

## 04-error-state.png
**Cél:** Adatbázis hiba esetén barátságos visszajelzés.
**Interakciók:** "Próbáld újra" gomb.
**Állapotok:** 500 Internal Server Error (szimulált).
**Hivatkozások:** US-05/AC1
